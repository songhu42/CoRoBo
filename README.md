#CoRoBo Project 
----
## *  Project Desc 
### .협동 작업 로봇 (Collaborative Working Robot)  
####  서봇에 로봇팔로 구성된 2대의 로봇으로, 하나의 로봇이 수행하지 못하는 대형 물체를 협력하여 이동 거치하는 로봇 제어 프로젝트. 

---
## * 폴더 구조  
### .src : 프로젝트 소스 파일  
### .docs : 각종 계획/실행 문서 파일 
### .study : 각종 참고용 스터디 자료 모음(영상, 링크, 문서 등)  
### .data : 참고용 데이터성 자료 모음
